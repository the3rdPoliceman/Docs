Prism
What does prism do?
Prism is a client side javascript component that formats code blocks in HTML pages. You specify a language for some code using css classes, and prism will then highlight the syntax accordingly. As shown below.

@import url(http://fonts.googleapis.com/css?family=Questrial);
@import url(http://fonts.googleapis.com/css?family=Arvo);
@font-face {
src: url(http://lea.verou.me/logo.otf);
font-family: 'LeaVerou';
}
