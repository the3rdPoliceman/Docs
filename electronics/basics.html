Relays, Solenoids & Contactors
When would you use a contactor rather than a relay? If you're using something that needs a lot of power e.g. above 240V 5A you'd normally use a contactor. Probably should learn more before deciding that though
Choosing a relay. Recommended Relays 4RD 931 410-05 or similar (available on aliexpress). Recommended by devin vanderhoof for car stuff. Reason for recommendation - skirted, watertight, nice terminator setup
Relays
  • Eletromechanical Relays - use an electromagnetic coil to attract a movable contact. Often have multiple sets of contacts, so one input change can utrn on/off multiple decvices
  • Solid State Relays - Can activate with very little power. Mo movable parts = longer life than electromechanical. Switch faster
