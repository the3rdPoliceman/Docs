Updating the Firmware
  • Locate the firmware you want
  • Copy the bin file to the route of the SD Card
  • Load the SD card into the printer and restart it
  • While restarting it will automatically do the firmware update

Troubleshooting
  • The print speed is fucking up, and goes really high even though I set it low - This happens because of a bug. If the name of the gcode file is too long (more than about 20 characters) it messes with the print speed
  • First layer Not Sticking
    • Clean the bed with alcohol, to remove the oil from your filthy mitts
    • A brim helps, especially for things with small footprints
    • Bump initial layer temp 10 degress higher than the rest
    • Lower initial layer height to maybe 90% of standard layer height
    • Slow down initial layer speed & initial layer travel speed
    • Bump initial layer line width to 110-120%
    • Turn off fan for the first layer
    • Print outer walls before inner
    • Bottom pattern to concentric & connect top/bottom polygons on
    • CHEP - Fixing First Layers on 3D Prints
    • Thomas Sanladerer has good example prints to show issues
