Navigation
  • Same across pages
  • Click on logo goes home
  • There is an indication of current location on the nav bar e.g. underline, color
  • Not too many items
  • If hierachy complexity needs it, Breadcrumbs (normally below main navigation) are useful
  • If part of a multi step process like airline booking, a wizard guide showing steps and indicating current step
