Spindle Speeds
One suggested rule from Woodturners Catalog gives
  • 150000/D(mm) = RPMmin
  • 225000/D(mm) = RPMmax
This would give the following speeds
Diameter RPMmin RPMmax
25mm 6000 9000
40mm 3750 5650
50mm 3000 4500
Vibration

The main cause of vibration in a wood lathe is an unbalanced workpiece, which is likely at the start. This is exacerbated at higher speeds, so ideally you want to be turning fast enough for a good finish, but not so fast as to increase vibration. This would suggest the following potential tactics

  • A rough shaping pass should be done, followed by one or two finer passes (My idea)
  • If speed is variable, the roughing pass could be done at lower speed than the finish pass (idea from https://blog.woodturnerscatalog.com/2012/08/safe-wood-lathe-speeds/

Idea: Would be interesting to know if some sort of vibration sensor cuold be used. Initially, while spinning but not yet cutting to determine balance. While cutting to detect poor cutting or chatter

Question: Could a large centered weight e.g. a heavy steel disc or heavy headstock, help to reduce vibration

Spindle Spped Variantion should be investigated. See https://www.youtube.com/watch?v=dXR2wDDFFm4 for an example and investigate further

Sanding Attatchment
Sanding Belt Options
  • 25*2mm belts
  • 25*50m belts
  • 762*25mm belts

Short belt option e.g. 762*25mm could be held in a zigzag along a set of rollers. Advantage being the simplicity of the setup.

Long belts roll from one pulley onto another. This would be more complicated mechenism, but longer time between replacement

For round pieces, belts would pun perpendicular to lathe. Similar setup could be used on router lathe to sand pieces with hex or octagon proflies
