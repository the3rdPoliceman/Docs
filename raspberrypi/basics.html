Connecting via SSH, and other useful stuff
  • SSHing - Connect to the same network as the pi. Run arp -a to list all the computers on the network, and identify the ip address of the pi. Connect by running ssh pi@<pi-ip-address>
  • Grabbing a file from the pi - Connect to the same network as the pi and run scp pi@192.168.0.139:/home/pi/Desktop/output.jpg /Users/dave/Documents
Diagrams
Physical Connections
Raspberry Pi 3B V1.2
  • 5V Micro USB power, recommended 2.5A
  • CSI camera port for Pi camera
  • DSI display port for Pi touchscreen display
  • 40-pin extended GPIO
  • 4 USB 2 ports
  • Full size HDMI
  • Micro SD port
  • 100 Base Ethernet
  • 4 Pole stereo output and composite video port
Raspberry Pi Zero W V1.1
  • Micro USB power 5v, minimum 1.2A, recommended 2.5A
  • Mini CSI camera connector
  • HAT-compatible 40-pin header
  • Micro USB On-The-Go (OTG) port
  • Mini HDMI port
  • Micro SD port
  • Composite video and reset headers
Specs & Non Physical Connections
Raspberry Pi 3B V1.2
  • Quad Core 1.2GHz Broadcom BCM2837 64bit CPU
  • 1GB RAM
  • BCM43438 wireless LAN
  • Bluetooth Low Energy (BLE)
Raspberry Pi Zero W V1.1
  • 1GHz, single-core CPU
  • 512MB RAM
  • 802.11 b/g/n wireless LAN
  • Bluetooth 4.1
  • Bluetooth Low Energy (BLE)
