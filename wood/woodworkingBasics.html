Markings
Symbols
Face Side: pigs tail with root where Face Side meets Face Edge
Face Edge: An arrowhead (V), with the tip pointing where Face Side meets Face Edge

So, the full marking consists of the tip of the arrowhead meeting the root of the pig tail
Marking Tips
When using a square, marking gauge etc. to mark, always mark with the fence of the marking tool along either the face side or face edge
Layout
Numbering Joints
Start from top left, and number the joints clockwise by writing the number on each joint piece

Laying Out A Frame
All Face Sides of a frame should be in the same plane i.e. all to the inside face of the door, or all to the outside face
All Face Edges should be to the inside of the frame i.e. to the centre of the glass of a window in a window frame


Laying Out A Case Or Box
All Face Edges should be in the same plane, to either the top or bottom of the box
All Face Sides should be to the inside of the box
