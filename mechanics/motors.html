Label Fields
  • HP - Horse power
  • KW - Kilowatts
  • HZ - Frequency, either 50 or 60Hz. No problem running on different frequency, it will just spin proportionally faster/slower.
  • V - Voltage expected. Can be considered rough i.e. 210 will be fine at 240 etc.
  • PH - How many phases
  • A - Current in amps.
  • CODE - Measure of the inrush (startup) current. When the motor starts, it draws a lot more current than when it is up to speed.
  • SF - How much load the motor can handle temporarily, as a ratio of the motor power
  • FR - Frame size. Can be looked up
  • AMB - Ambient temperature suitable for running the motor
  • NEMA DESIGN - Refers to torque profile e.g. if motor starts with very high torque,
  • TIME RATING/DUTY - e.g. Cont for continuous. How long you can run the motor at full load
  • ENC - Enclosure type

Universal Motor
Types
Main features
  • Similar to a DC motor, but instead of permanent magnets, electromagnets are used
  • Can run on AC or DC
DC Motor
Types
  • DC Permanent Magnet motors - Have permanent magnets in the stator. The brushes contact the coils in turn as the motor rotates (via the commutator), gradually changing the magnetic field position in the rotor. This change in the field position puts a force on the rotor from the stator magnets
Main features
AC Motor
Types
Main features
  • Can be used with a variable-frequency drive (VFD). This type of motor drive can be used to control AC motor speed and torque by varying motor input frequency and, depending on topology, to control associated voltage or current variation
Induction Motor
Types
Main features
  • Run on AC
  • Often require a capacitor, or even 2 for starting
Identifying a motor
Clues from the label
  • DCPM - DC Permanent Magnet
  • DC - DC Motor
  • See Hz and Phase(PH) on the label? - Probably an induction motor
  • Speed - if speed is a bit less than 3600, 1800, 1200, or 800 @60hz, or 3000, 1500, 1000, 750 @50Hz it's probably an induction motor. These speeds are the no load (and no friction) speeds of motors with 2,4,6 and 8 poles, and the motor speed on the label is typically around 5% less e.g. 1725 instead of 1800
Clues from inspection
  • See brushes on the commutator? - either a universal motor or a DC motor. So which is it? Universal motor has an electromagnet stator field
  • Is it magnetic? Most likely a permanent magnet DC motor
  • See a squirrel cage rotor inside (no electrical connection to rotor)? Probably induction. A squirrel-cage rotor consists of a cylinder of steellaminations, with aluminum or copper conductors embedded in its surface, and resembles a rodent exercise wheel.
