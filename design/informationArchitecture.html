Information Architecture
Structuring a website

The pages that make up a website need to be in some sort of a logical hierarchy. This is known as information architecture.

Navigation Design
  • Navigation should be consistent across pages
  • There is an indication of current location on the nav bar e.g. underline, color
  • Clicking on the company logo goes home
  • Not too many items
  • If hierachy complexity needs it, breadcrumbs (normally below main navigation) are useful
  • When in a multi step process like flight booking, a wizard guide showing steps and indicating current step is helpful
