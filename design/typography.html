Font Choice & Setup
Choose your Font
Using one font keeps life simpler - two types needs skill. If you really feel you need two, try using two fonts designed to be used together e.g. Circe and Circe Rounded
How to a choose font face?
  • Decide based on popularity (Google fonts can tell you what is popular)
  • Steal from sites
  • It's a good idea to use something with at least 5 weights (not because you should use 5 weights, but because it means care was taken when designing it)
Google Fonts, or fonts in use are also good places to find fonts that go well together
Select a readable size for main text
Choose the most readable font size for the main body of the text
Normally this is 16 or 18 pixels
Then create a type scale
Starting from your base font, create a type scale (It's probably better to generate this by hand rather than use online generators, as they will end up giving fractional font sizes)
Stick to px or rem Font size units rather than em — it’s the only way to guarantee you’re actually sticking to the system

Online options include grid lover and type-scale.com

Set up your line height
Define in em
Typically, you will increase line height with small font sizes to make text more readable e.g. 1.5 - 1.6em
As font sizes increase, you will want a relatively smaller line height - Can be as low as 1em
It is especially likely to need to be decreased if the text is all caps, as the lack of descenders makes the gap between lines appear huge
Don't fuck with the letter spacing....mostly
A possible exception is when USING UPPERCASE.
Often, a bit of extra letter spacing can help readability when USING UPPERCASE
Text Layout Rules
Alignment & justification
Vertical Alignment - Align text by baseline
Horizontal Alignment or Justification - Left align almost everything. Right align numbers. Don't justify text on both sides, you'll just end up with random gaps in the text.
Centre align is generally a bad idea, but can work for small amounts of text in certain situations
Stay away from dark corners and dangerous edges
Don't have text jammed right into the edges, or up against other components. It's weird.
